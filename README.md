# M32 en Logisim Evolution

Implementación del procesador educativo M32 en Logisim Evolution, desarrollado como proyecto de tesis para la carrera de Ingeniería Civil en Informática en la Universidad del Bio-Bio

## Descripción

El M32 es una arquitectura de procesador educativa de 32 bits diseñada originalmente por el profesor Luis Mateu de la Universidad de Chile. Este proyecto implementa una simulación completa del M32 en Logisim Evolution, permitiendo ejecutar y visualizar el funcionamiento de programas en esta arquitectura.

### Características principales
- Arquitectura RISC de 32 bits
- Visualización paso a paso de la ejecución
- Herramientas para la construcción de programas en ensamblador para esta arquitectura

## Requisitos

- Logisim Evolution 3.9.0
- Java 21 o superior
- Sistema operativo: Windows, Linux o macOS

## Instalación y ejecución

1. Descarga el último lanzamiento disponible desde el menú de [Releases](http://github.com/fferradamerino/m32logievo/releases).
2. Extrae el archivo ZIP en el directorio de trabajo deseado.
3. Ejecuta el script correspondiente a su sistema operativo:
   - **Linux/macOS**: `./run.sh`
   - **Windows**: `run.bat`
